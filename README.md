# ComputerGraphicsProject
This repository contains additional material for the Computer Graphics course project.

Contents:
```
.
├── GS_kaggle.ipynb
├── filter.py
├── poisson_disk.py
└── sugar_kaggle.ipynb
```
The repository contains the notebooks used to run gaussian splatting and SuGaR on kaggle.
The filter.py and poisson_disk.py files are used for processing the gaussian splatting output and extracting a mesh.
